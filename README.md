# 📊 Amazon Employee Dashboard | Tableau

## 📌 Overview

This project presents an interactive Tableau dashboard for analyzing employee and salary data across departments and countries.

## 📈 Key Metrics

* Total Employees: 689
* Total Annual Salary: 17,099,892
* Average Salary: 2,068

## Dashboard Features

* Employee growth over years
* Average salary by department
* Employee distribution by country
* KPI cards for workforce analysis

## Tools Used

* Tableau
* Data Visualization
* Dashboard Design
* Data Analysis

## Key Insights

* Employee count peaked in 2019.
* Salary levels vary across departments.
* Workforce distribution differs by country.
